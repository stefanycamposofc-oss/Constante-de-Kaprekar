# Rotina de Kaprekar

Implementação da Rotina de Kaprekar para números de 4 dígitos em Python.

## O que é?

A Rotina de Kaprekar é um processo matemático que, dado qualquer número de 4 dígitos (que não seja um repdigit), reorganiza seus dígitos em ordem crescente e decrescente, subtrai o menor do maior, e repete o processo com o resultado. Esse processo sempre converge para o número **6174** em no máximo 7 iterações.

## Como funciona o programa?

1. O usuário digita um número de 4 dígitos
2. O programa valida a entrada
3. A cada iteração, organiza os dígitos e faz a subtração
4. Repete até atingir a constante **6174**
